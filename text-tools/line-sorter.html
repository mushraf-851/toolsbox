How does the line sorter work?

Paste any list — one item per line — and choose a sort mode. The tool sorts entirely in your browser using JavaScript's built-in localeCompare for alphabetical modes and Fisher-Yates for random shuffle. Everything runs instantly with no data sent to a server.

Natural number sort understands that "Item 2" comes before "Item 10", unlike a plain alphabetical sort that would put "10" before "2". It is useful for numbered lists, file names, and versioned items.

Frequently asked questions

How do I sort lines alphabetically online?
Paste your list, select "A to Z" mode and click Sort Lines. Lines are sorted in ascending alphabetical order using locale-aware comparison that correctly handles accented characters.
Can I sort lines in reverse alphabetical order?
Yes — select "Z to A" mode to reverse-sort all lines in descending alphabetical order. "Reverse" mode flips the existing line order without sorting alphabetically.
Can I shuffle lines randomly?
Yes — select "Random Shuffle" mode. The tool uses the Fisher-Yates shuffle algorithm which gives a perfectly uniform random distribution. Click Sort again to re-shuffle.
Can I sort and remove duplicates at the same time?
Yes. Enable "Remove duplicates" and duplicates are removed before sorting. The stats bar shows how many duplicate lines were removed.
Is the sort case-sensitive?
By default sorting is case-insensitive, so "Apple" and "apple" sort together. Enable "Case-sensitive sort" to treat uppercase and lowercase letters as distinct characters for ordering.
